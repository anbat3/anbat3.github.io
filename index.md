# Hi, I'm Anabel 👋

## Computational Linguist | NLP & AI Specialist

I used to break down texts for top publishers; now I build and deploy the code that helps machines understand them. After 4 years managing language quality and content data structures in digital publishing, I shifted my focus to NLP and conversational systems.

Today, my day-to-day involves training NLU pipelines, orchestrating GenAI agents with CrewAI, and building RAG architectures. Coming from a strong linguistic background means I don’t just write prompts or code models; I know exactly how syntax behaves, how context changes meaning, and how to systematically mitigate biases to ensure high-quality outputs.
